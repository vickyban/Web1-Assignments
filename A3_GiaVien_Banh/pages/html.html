What is HTML?

HTML is the backbone of all web pages. It stands for HyperText Markup Language,which controls the content of Web pages.

HTML tags

In HTML, the contents are wrapped inside <tags> elements. They tell the browser how the content between the tags should be displayed, and also provide hints to other web developers of what the content is about

Normally, the tags come in pairs, opening and closing tag. However, there are also empty elements, which do not have content.

Examples:

  • <header>header</header>

  • <p>paragraph</p>

  • <img> to link images

Basic structure of a HTML

Because of different versions of HTML, every html file begins with something like < !DOCTYPE html> to tells the browser what version of HTML the document is written in.

This is followed by <html></html>. Inside html tag contains two elements: head and body tags. The information about the document (such the title, author, language usage) are documented in the head. They are not visible to the web page. The actual contents that are visible of the web go into the body tags

<!DOCTYPE htm> <html>
<head>
<title>Webpage Name</title>
</head>
<body>
webpage content </body>
</html>
