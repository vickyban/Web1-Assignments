Filename convention:

Practicing good file management facilitates the development of web pages and the expansion of the web pages. One step is to have good filename system. By convention, filenaming should :

  • Avoid spaces in filenames
  • Use only Lowercase letters
  • Avoid special characters
  • Use short and descriptive name
  • Use Carmel case if the name is made out of serval words
  • Start with underscore if it is an asset folders
  • Name index.html for the main page
  • Be consistent with file extensions
