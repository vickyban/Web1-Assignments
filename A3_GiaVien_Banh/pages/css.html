What is CSS?

CSS stands for Cascading Style Sheets. It describes to the browser how to to display HTML elements.

How to apply CSS to HTML?

There are three ways of adding style sheets to HTML document:

  • External style sheet, stores CSS document into a seperate file and links it to HTML using the syntax
    <link rel="stylesheet" type="text/css" href="path to css file.css">
  • Internal style sheet, the information is defined inside the <head> section of the HTML.
  • Inline style sheet, applies style to single HTML element in the style attribute

In general, having CSS separate from the HTML content provides more flexibility and efficiency in managing and alterting style sheets. Hence, External style sheet is more prefered.

Media Queries

Media Queries allows to control multiple style sheets based on some conditions. If a condition is met, a specific style will be applied. This gives web pages the responsiveness, enables it to adapt its content to fit all devices.There are two ways to apply Media Queries:

  • @media in CSS file to make changes in the file.
  • adding media attribute inside the <link> to refer to different stylesheets.
